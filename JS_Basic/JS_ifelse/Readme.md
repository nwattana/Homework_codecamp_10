# แบบฝึกหัด
## EX1. Browser จะโชว์ข้อความอะไร

``` JAVASCRIPT
    if(0){
        alert('zero');
    }else {
        alert('Noo');
    }
```

## Ex.2 ใช้ if else ในการเขียนถามชื่อของคุณ
- ถ้าตอบถูกให้แสดงคำว่า “เก่งมาก”  
- ถ้าตอบผิดให้แสดงคำว่า “คุณไม่รู้จักชื่อฉัน”
